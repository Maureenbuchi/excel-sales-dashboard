# Excel-Sales-Dashboard
## Project Overview
This project is an Excel-based sales dashboard I built during my Data Analysis training.
It covers the complete process from data cleaning to visualization, using Excel tools like Pivot Tables, Pivot Charts, and Slicers to generate insights and make the data easier to understand.

This dashboard was created in class with guidance from my tutor, and it represents one of my foundational steps in becoming confident with Excel for analytics.



## Project Objective

- To learn how to organize and clean a dataset in Excel

- To build a fully interactive dashboard using Pivot Tables

- To understand how slicers improve filtering and usability

- To gain confidence in turning raw data into visual insights

- To practice creating reports that communicate information clearly

  
 ## Tools & Skills Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Basic Data Transformation
- Data Visualization
-Dashboard Design
